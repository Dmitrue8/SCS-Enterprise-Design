# Проектирование структурированной кабельной системы предприятия

## Описание проекта
Учебный инженерный проект по проектированию структурированной кабельной системы (СКС) для производственного предприятия.  
Проект охватывает полный цикл проектирования СКС: от анализа требований и выбора архитектуры до расчёта кабельных трасс, подбора оборудования и подготовки проектной документации.

## Цель проекта
- Проектирование надежной и масштабируемой СКС предприятия  
- Соблюдение требований отраслевых стандартов  
- Подготовка проектной документации, пригодной для практической реализации  

## Что было сделано
- Анализ требований к сетевой инфраструктуре  
- Выбор архитектуры СКС (иерархическая «звезда»)  
- Проектирование магистральной и горизонтальной подсистем  
- Расчёт длины кабельных трасс и резервов  
- Подбор кабельной продукции и оборудования  
- Проектирование телекоммуникационных шкафов и кроссов  
- Подготовка кабельного журнала  
- Формирование спецификации оборудования  
- Оформление проектной документации  

## Используемые технологии и стандарты
Сетевые технологии:
- Ethernet  
- Структурированные кабельные системы (СКС)

Кабельная инфраструктура:
- Витая пара Cat.6  
- Оптоволоконный кабель (Single Mode, OS2)

Стандарты и нормативы:
- ГОСТ 53246-2008  
- ISO/IEC 11801  
- TIA/EIA-568
